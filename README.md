\# UserAuth-C-System



A file-based user registration system written in C.



\## Features

\- Unique username validation

\- Password length validation

\- Basic Caesar cipher password encryption

\- Persistent storage using file handling



\## Tech Used

\- C

\- GCC

\- File Handling

\- Structs



\## How to Run



Compile:

gcc main.c -o userauth



Run:

./userauth



\## Note

Passwords are encrypted using a simple Caesar cipher.

This project is for learning purposes and does not implement real-world secure hashing.
